# FitTrack 健身追蹤應用程式文件

## 1. 專案介紹
FitTrack 是一款 iOS 健身追蹤應用程式，使用 Flutter 框架開發。本應用程式旨在為健身愛好者提供簡單直觀的運動記錄工具，幫助使用者追蹤自己的健身進度。

### 1.1 專案目標
- 提供直觀且易用的健身記錄介面
- 建立符合 Apple Human Interface Guidelines 的使用者體驗
- 確保應用程式的可擴展性與維護性
- 打造高效能且穩定的移動應用程式

### 1.2 技術棧
- 框架：Flutter
- 程式語言：Dart
- 目標平台：iOS
- 設計規範：Apple Human Interface Guidelines
- 版本控制：Git

## 3. 開發方式

### 3.1 開發環境設置
```bash
# 必要條件
- Flutter SDK: 3.0.0 或以上
- Xcode: 13.0 或以上
- iOS Simulator 或實體 iOS 設備
- Git

# 環境設置步驟
1. 安裝 Flutter SDK
2. 設置 iOS 開發環境
3. 克隆專案儲存庫
4. 安裝專案依賴
```

### 3.2 開發規範
1. **代碼風格**
   - 遵循 Flutter 官方代碼風格指南
   - 使用 dartfmt 進行代碼格式化
   - 確保代碼可讀性與一致性

2. **命名規範**
   - 檔案名稱：小寫，下劃線分隔（例：workout_card.dart）
   - 類名：大駝峰命名法（例：WorkoutCard）

3. **註釋規範**
